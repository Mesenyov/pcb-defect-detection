# 🔍 Адаптивный подход на основе сиамских нейронных сетей для обнаружения дефектов печатных плат

![Project Status](https://img.shields.io/badge/status-active-green)
![Python](https://img.shields.io/badge/python-3.8%2B-blue)
![Framework](https://img.shields.io/badge/framework-PyTorch%20%7C%20FastAPI-red)
![License](https://img.shields.io/badge/license-MIT-grey)

Официальный репозиторий для научной статьи: **"Siamese Neural Network-Based Adaptive Approach for Open-Set Defect Detection and Classification in Printed Circuit Boards"**.

Данный проект представляет собой фреймворк автоматизированного визуального контроля (AOI) печатных плат (PCB). Система предназначена для высокоточного обнаружения производственных дефектов путем сравнения тестируемого изображения с эталоннным.

**Ключевая особенность:** В отличие от традиционных детекторов объектов, ограниченных парадигмой закрытого множества, данная система способна обнаруживать ранее неизвестные типы дефектов без необходимости полного переобучения модели, достигая метрики **OOD F1-Score равной 83.2%**.

## 🖥️ Интерфейс системы

![Веб-интерфейс детекции дефектов печатных плат](static/img/screenshot.png)
> **Интерактивная аналитическая панель:** Веб-интерфейс демонстрирует полный цикл локализации и метрической классификации дефектов. Система выделяет известные дефекты точными ограничивающими рамками (bounding boxes), отображает уверенность модели (Confidence) в режиме реального времени на основе латентных расстояний ArcFace, а также явно маркирует новые неизвестные аномалии.

## 📊 Результаты экспериментов (End-to-End оценка OOD)

Предложенный фреймворк сравнивался с современными детекторами закрытого множества (семейство YOLOv8) и детектором открытого мира (ORE) на пользовательском валидационном датасете, содержащем **принципиально новые аномалии** (царапины, окисление, загрязнения и др.).

## 🛠 Архитектура

Фреймворк разделяет задачи локализации аномалий и метрической классификации на легковесный двухэтапный конвейер (pipeline):

### Этап 1: Локализация аномалий (Siamese U-Net)
Выявляет структурные отклонения от эталонного шаблона с использованием дифференциального вычитания на уровне признаков.
*   **Backbone:** Легковесная сеть **MobileNetV3-Large**.
*   **Механизм:** Сиамские ветви с общими весами извлекают признаки на трех иерархических масштабах. Абсолютные разности признаков ($D_i$) подаются в декодер с skip-connections.
*   **Обучение:** Двухфазный режим с использованием **BCE + Dice Loss** для обеспечения точного определения границ дефектов и подавления дисбаланса классов.

### Этап 2: Метрическая классификация (ProtoNet)
Классифицирует локализованные дефекты или отклоняет их как новые (OOD) аномалии.
*   **Backbone:** Вторичный энкодер **MobileNetV3-Large**.
*   **Стратегия:** **Scale-recovery cropping** — дефекты вырезаются из исходного изображения высокого разрешения на основе масок из Этапа 1, чтобы сохранить мельчайшие морфологические детали.
*   **Метрическое обучение:** Оптимизируется с помощью функции потерь **ArcFace** для создания высокодискриминативных латентных кластеров на единичной гиперсфере.
*   **Детекция OOD:** Используется **стратегия адаптивного порога** (правило $3\sigma$) с аналитически оптимизированным глобальным минимальным порогом ($T_{min}$) для изоляции объектов вне распределения.

## 🚀 Установка и запуск

### Требования
*   Python 3.8+
*   Git LFS (требуется для загрузки весов модели)

### 1. Клонирование и загрузка весов
```bash
git clone https://github.com/Mesenyov/pcb-defect-detection.git
cd pcb-defect-detection
git lfs pull
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Запуск веб-сервиса
```bash
python main.py
```
FastAPI приложение будет доступно по адресу: `http://localhost:8000`. Вы можете загрузить тестируемую плату и эталон, чтобы увидеть результаты работы пайплайна в UI.
