# 🔍 PCB Defect Detection System (Siamese U-Net + ProtoNet)

![Project Status](https://img.shields.io/badge/status-active-green)
![Python](https://img.shields.io/badge/python-3.11-blue)
![Framework](https://img.shields.io/badge/framework-PyTorch%20%7C%20FastAPI-red)
![License](https://img.shields.io/badge/license-MIT-grey)

Программный комплекс для автоматической оптической инспекции (AOI) печатных плат. Решение направлено на обнаружение производственного брака методом сравнения с «Золотым эталоном» (Golden Template).

**Ключевая особенность:** Система способна детектировать **неизвестные типы дефектов (Anomaly Detection)**, отсутствующие в обучающей выборке, благодаря использованию Metric Learning и функции потерь ArcFace.

---

## 📊 Результаты и Сравнение

Наше решение сравнивалось с современным детектором объектов **YOLOv8m**. Основной метрикой выбрана **Recall (Полнота)**, так как в задаче контроля качества критически важно минимизировать пропуск брака.

| Модель | Recall (Полнота) | Precision (Точность) | F1-Score | Способность к выявлению неизвестных дефектов |
| :--- | :--- | :--- | :--- | :--- |
| **YOLOv8m** (Baseline) | 93.4% | 97.1% | 0.95 | ❌ Нет |
| **Siamese U-Net** (Наше решение) | **98.5%** 🏆 | **97.5%** | **0.98** | ✅ **Да (Zero-shot)** |

> **Вывод:** Разработанная система пропускает в 4 раза меньше дефектов, чем стандартный детектор, и обладает гибкостью к изменению типов брака.

## 🛠 Архитектура

Система состоит из двух последовательных модулей:

### 1. Детектор аномалий (Siamese U-Net)
Отвечает за локализацию любых отклонений от эталона.
*   **Backbone:** EfficientNet-B0 (предобученный на ImageNet).
*   **Метод:** Сиамская сеть с общими весами. Вычисление абсолютной разности признаков (Feature Difference) на трех масштабах.
*   **Loss:** Комбинация `BCEWithLogitsLoss` + `DiceLoss` для работы с дисбалансом классов (дефекты занимают <1% площади).

### 2. Классификатор (Metric Learning / ProtoNet)
Отвечает за определение типа дефекта (или пометку его как "Неизвестный").
*   **Подход:** Prototypical Networks.
*   **Training:** Обучение с функцией потерь **ArcFace**. Это позволяет сжать классы в компактные кластеры на гиперсфере.
*   **Inference:** Классификация по косинусному расстоянию до центроидов (прототипов) классов. Если расстояние превышает порог — дефект помечается как `Unknown`.

## 🚀 Установка и Запуск

### Предварительные требования
*   Python 3.8+
*   Установленный Git LFS (для загрузки весов моделей)

### 1. Клонирование репозитория
```bash
git clone https://github.com/Mesenyov/pcb-defect-detection.git
cd pcb-defect-detection

# Загрузка весов моделей (обязательно!)
git lfs pull

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Запуск веб-сервиса
```bash
python main.py
```
Сервис будет доступен по адресу: `http://localhost:8000`
